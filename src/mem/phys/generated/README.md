# Generated First-Light ROM Data

These files are generated by `tools/bake_rom_image.py` and committed on purpose.

- `alu_loop_rom_data.spade` is baked from `bench/roms/out/alu_loop.gb` at `--size 1024`
- `bg_static_rom_data.spade` is baked from `bench/roms/out/BG_STATIC.gb` at `--size 1024`
- `joypad_bg_smoke_rom_data.spade` is baked from `bench/roms/out/joypad_bg_smoke.gb` at `--size 2048`

The source `.gb` artifacts are padded to `32768` bytes by `rgbfix`. The bake step truncates
them down to the actual first-light envelope and records deterministic CRC32/SHA256 metadata
alongside the emitted `[uint<8>; N]` literal.
